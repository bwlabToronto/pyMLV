Open Affective Standardized Image Set (OASIS)

The Open Affective Standardized Image Set (OASIS) was released by Benedek Kurdi, Shayn Lozano, and Mahzarin R. Banaji in 2016

The images, code and several sets of ratings of the images are available at: https://osf.io/6pnd7/

The paper that introduces OASIS is:

Kurdi, B., Lozano, S., & Banaji, M. R. (2017). Introducing the Open Affective Standardized Image Set (OASIS). Behavior Research Methods, 49(2), 457â€“470. http://doi.org/10.3758/s13428-016-0715-3

The line drawings were generated by automatically detecting and tracing contours using the MLV Toolbox.
